new(options={})

Creates a new howcast client to interact with the Howcast API

Inputs

Options include:

  • key — REQUIRED - API key: obtainable from www.howcast.com/api_keys/new

Exceptions

  • Howcast::ApiKeyNotFound — raised if the options[:key] value is nil

guide_search(query, options = {})

Provides access to the Howcast wiki guide search API.

See video_search documentation for inputs, outputs and exceptions

Examples

Get the first page of howcast wiki guides matching ‘poker’. 

  Howcast::Client.new.guide_search("poker")

Get the third page of howcast wiki guides matching ‘traveling‘ 

  Howcast::Client.new.guide_search("traveling", :page => 3)
guides(options = {})

REFACTOR: Consider combining with videos method Provides access to the Howcast list guides API.

Inputs

The options are:

  • :page — The page number to retrieve (defaults to 1). There are 16 guides per page.
  • :sort — One of most_recent (default) | most_viewed | top_rated | most_emailed | most_discussed | top_favorites
  • :filter — One of all | featured | howcast_written (default) | community_written

Outputs

An array of guide objects

Exceptions

  • Howcast::ApiNotFound — raised if the requested sort and filter is malformed or not available (404)

Examples

Get the first page of most recent howcast written guides. 

  Howcast::Client.new.guides

Get the third page of top favorites which are featured 

  Howcast::Client.new.guides(:page => 3, :sort => "top_favorites", :filter => "featured")

video_search(query, options = {})

Provides access to the Howcast video search API.

Inputs

  • query — The string query which you want to search for

The options are:

  • :page — The page number to retrieve (defaults to 1). There are 10 videos per page.
  • :mode — Mode to search, using :extended will allow title:something searches

Outputs

An array of video objects

Exceptions

  • Howcast::ApiNotFound — raised if the requested sort and filter is malformed or not available (404)
  • ArgumentError — raised when the required 1 argument isn‘t supplied

Examples

Get the first page of howcast videos matching ‘poker’. 

  Howcast::Client.new.video_search("poker")

Get the third page of howcast videos matching ‘traveling‘ 

  Howcast::Client.new.video_search("traveling", :page => 3)

attach_api_key(url)

Appends the api key to a url and returns the appended url

Inputs

  • url — the url to append the api_key to

Outputs

The url with the api_key appended

Examples

  attach_api_key("http://www.howcast.com")
  => "http://www.howcast.com?api_key=APIKEYHERE"

  attach_api_key("http://www.howcast.com/videos/most_recent/all?page=2")
  => "http://www.howcast.com/videos/most_recent/all?page=2&api_key=APIKEYHERE"
establish_connection(relative_path)

Establishes a connection with the Howcast API. Will auto append the api key, that was used to instantiate the Client object, to the URL

Inputs

  • relative_path — should be the path after www.howcast.com/

Outputs

Hpricot object encapsulating the xml returned from the Howcast API

Exceptions

  • Howcast::ApiNotFound — raised if the requested relative_path is malformed or not available (404)
  • Howcast::ApiKeyNotFound — raised if the api_key is invalid

Examples

Get the Hpricot data for most recent howcast studios videos 

  establish_connection("videos/most_recent/howcast_studios.xml")
parse_single_xml(xml, klass)

Parses the xml for a single item from xml and creates a new klass object

Inputs

  • xml — See below for a sample xml input
  • klass — Class to create - Guide | Video supported

Sample input xml 

  <guide>
     <id>1086</id>
     <rating>96</rating>
     <title>How To Choose a Paintbrush</title>
     <category-id>19</category-id>
     <description>
        <![CDATA[Yes, you could just use your fingers, but selecting the best brushes for your painting might achieve a slightly more grown-up result.]]>
     </description>
     <views>362</views>
     <permalink>http://www.howcast.com/guides/1086-How-To-Choose-a-Paintbrush</permalink>
     <username>michaelrsanchez</username>
     <created-at>Thu, 20 Dec 2007 14:14:58 -0800</created-at>
   </guide>

Outputs

klass object with initialized attributes
